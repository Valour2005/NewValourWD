# Portfolio Website

A colorful professional portfolio website built with Bootstrap 5.

## Files included
- `index.html` — main landing page with Bootstrap layout
- `styles.css` — custom styling and gradients
- `script.js` — lightweight JavaScript loader

## Usage
1. Open `index.html` in your browser.
2. Customize the text, colors, and project details to match your personal brand.
3. Replace the placeholder contact email with your real email.

## Notes
- Uses Bootstrap 5 for responsive layout and navigation.
- The site includes polished section cards, gradient accents, and a colorful dark theme.
- You can extend this with more project cards, testimonials, or case study pages.
