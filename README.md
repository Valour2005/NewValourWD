# Portfolio Website

A polished professional portfolio website scaffolded for a clean presentation.

## Files included
- `index.html` — main landing page
- `styles.css` — responsive design and visual styling
- `script.js` — mobile navigation toggle

## Usage
1. Open `index.html` in your browser.
2. Customize the text, colors, and project details to match your personal brand.
3. Replace the placeholder contact email with your real email.

## Notes
- This site is built with plain HTML, CSS, and JavaScript.
- It is easy to extend with additional sections, portfolio items, or animations.
